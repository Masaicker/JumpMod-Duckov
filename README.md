# JumpMod - Duckov 跳跃增强模组

Escape from Duckov 游戏的跳跃功能增强模组，提供可配置的跳跃控制。

## 功能特性

- 增强的跳跃机制，支持自定义参数
- 可配置的跳跃高度和灵敏度
- 调试日志功能，便于排查问题
- 与原游戏无缝集成

## 安装方法

1. 下载最新版本
2. 解压到 Duckov/Mods 文件夹
3. 编辑 `JumpConfig.json` 配置文件

## 配置说明

编辑 `JumpConfig.json` 自定义：
- 跳跃倍数
- 摔落伤害修正
- 调试日志等级

## 依赖要求

- Escape from Duckov 游戏本体
- BepInEx 模组框架
- 0Harmony 库
