# JumpMod - Duckov 跳跃模组

Escape from Duckov 游戏的马里奥风格跳跃模组，支持按住跳跃键持续加速。

## 功能特性

- **马里奥式跳跃**：按住跳跃键可以持续提升跳跃高度
- **土狼机制**：离地后 150ms 内仍可跳跃
- **输入缓存**：落地前 200ms 内按下跳跃键，落地后自动跳跃
- **混合控制**：初始力度 + 持续微调的跳跃方式
- **完全可配置**：通过 ModSetting 系统调整所有参数

## 配置选项

使用 ModSetting 配置界面调整：
- **最小跳跃力度**：0.5-15.0
- **最大跳跃力度**：5.0-25.0
- **初始加速度**：1.0-30.0
- **加速度衰减**：0.1-0.95
- **跳跃按键**：默认 Z 键
- **调试日志**：开启/关闭

## 使用方法

1. 安装模组后，在游戏中使用配置的跳跃键（默认 Z）
2. 轻按跳跃键执行小跳
3. 按住跳跃键跳得更高
4. 支持连跳和快速跳跃

## 依赖要求
